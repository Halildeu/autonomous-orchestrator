North Star Workflow v1

	              
Canonical flow: Reference -> Assessment -> Gap -> PDCA

	              
    
                
  1. Reference scope: Theme/Subtheme set is approved (ACTIVE).
    2. 
                
  2. Criteria match: bind selected subtheme to default perspective criteria set.
    3. 
                
  3. Reference synthesis: per criterion, collect world trends/best practices and write readable summary.
    4. 
                
  4. Assessment synthesis: per criterion, map current-state evidence for the same subtheme.
    5. 
                
  5. Gap synthesis: derive deterministic differences (reference vs current state) per criterion.
    6. 
                
  6. Reading mode: use Lens Findings filters (stage + subject + theme + subtheme + criterion).
    7. 
                
  7. PDCA: prioritize closure actions, recheck, and track regression.
    8. 
	              

	              
Note: Workflow Stage is the single primary reading axis in Lens Findings.
