# Project Roadmaps (CORE‑etkilemeyen)

Bu klasördeki roadmap’ler **proje/müşteri akışları** içindir ve **CORE’e yazamaz**.
Core kilidi (core_lock) kapalıyken bu roadmap’ler yalnızca:
- `workspace_root` altında, ve
- (varsa) allowlist edilmiş `external_root` altında
yazım yapabilir.

Kural:
- Proje roadmap’leri CORE SSOT’u değiştirmez.
- “Neredeyiz/Sırada ne var?” için `project-status` (AUTOPILOT CHAT) kullanılır.

## Projeler
- PRJ-KERNEL-API — program-led Kernel API (v0.4.4 guardrails görünür; live gate default OFF) (roadmaps/PROJECTS/PRJ-KERNEL-API/roadmap.v1.json)
- PRJ-M0-MAINTAINABILITY — core refactor tracker (plan-only; core_lock protokolü gerekir) (roadmaps/PROJECTS/PRJ-M0-MAINTAINABILITY/roadmap.v1.json)
- PRJ-PORTFOLIO-TRACKING — program-led proje portfolyosu görünümü (doc-only v0.1) (roadmaps/PROJECTS/PRJ-PORTFOLIO-TRACKING/roadmap.v1.json)
